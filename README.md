## 1. Iniciar MySQL

Verificar que el servicio de MySQL se encuentre activo.

Ubicarse en la carpeta backend, con el comando `cd backend` y ejecutar:


Abrir el proyecto mediante Live Server desde Visual Studio Code.

Ingresar a:
